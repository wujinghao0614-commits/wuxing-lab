# 首页 (Home) 设计文档

> 最后更新：2026-03-16

---

## 📱 页面概述

- **页面名称**: 首页 (Home)
- **路径**: `/` 或 `index`
- **功能**: 展示用户今日能量状态概览，快速入口到核心功能

---

## 🎨 布局结构

```
┌─────────────────────────────┐
│         状态栏              │  ← 电池、网络、时间
├─────────────────────────────┤
│       头像 + 名字            │  ← 用户信息区域
│       金木水火土            │  ← 今日五行
├─────────────────────────────┤
│                             │
│     [今日能量仪表盘]        │  ← 核心：能量环形图
│        85分                 │  
│                             │
├─────────────────────────────┤
│  [记录]    [日历]    [分析] │  ← 快捷入口按钮
│                             │
├─────────────────────────────┤
│    ┌─────┐ ┌─────┐ ┌─────┐  │  ← 功能卡片
│    │五行 │ │运势 │ │辟谷│  │
│    │平衡 │ │提示 │ │指南│  │
│    └─────┘ └─────┘ └─────┘  │
│                             │
├─────────────────────────────┤
│        底部导航栏           │  ← 首页/记录/我的
└─────────────────────────────┘
```

## 🎨 视觉风格

- **主色调**: 
  - 主色: `#1A1A2E` (深蓝黑)
  - 辅色: `#16213E` (深蓝)
  - 强调: `#E94560` (朱红)
- **五行配色**:
  - 金: `#FFD700`
  - 木: `#00A86B`
  - 水: `#1E90FF`
  - 火: `#FF4500`
  - 土: `#DAA520`
- **字体**: 思源黑体 / Noto Sans SC
- **圆角**: 16px
- **阴影**: `0 4px 20px rgba(0,0,0,0.15)`
